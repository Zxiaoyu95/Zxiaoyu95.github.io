docker创建hadoop镜像

本文总共4320个字 | 您是第位看到它们的小伙伴

引言：docker镜像的构建对于程序开发者来说是十分友好的，由于在Dockerfile的过程踩了坑，下面就记录一下创建hadoop镜像的过程。

首先先创建一个文件命名Dockerfile

我们需要一个基础镜像，在这里我们有个alpine的小型镜像，里面支持JDK1.8

在同目录下添加几个文件

bootstrap.sh:镜像启动执行的脚本（ssh hadoop namenode -format，hadoop中username等）

hadoop-2.7.7.tar.gz:hadoop的下载包
hadoop-native-64-2.7.0.tar:hadoop 本地包
core-site.xml/yarn-site.xml/hdfs-site.xml:自己定义要替换的xml文件

Dockerfile内容

Dockerfile里面包含了ssh配置，hadoop配置（jdk，端口，配置文件等）

Dockerfile里面继承了好多命令（ADD RUN COPY ENV），ADD与COPY的区别是前者可以解压压缩包，以上实验需求记录的过程。
